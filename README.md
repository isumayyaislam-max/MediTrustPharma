# MediTrustPharma 
### Full-Stack E-Pharmacy & Medicine Inventory Management System

MediTrustPharma is a web-based full-stack e-pharmacy platform designed to streamline medicine browsing, real-time availability tracking, and secure order processing. The system connects a highly responsive user interface with an organized database architecture to simplify online pharmaceutical transactions and inventory tracking.

---

##  Key Features

* **Real-Time Availability Search:** Implements dynamic query functionality allowing users to instantly check medicine stock levels via precise backend SQL lookups.
* **Responsive Frontend Catalog:** Features a clean, accessible layout designed for seamless product navigation and structured browsing across multiple device types.
* **Relational Inventory Management:** Leverages a structured relational database model to track product specifics, categorize medical inventory, and manage order workflows efficiently.
* **Secure Order Processing:** Programmed with robust backend logic to capture user requests, process product selections, and handle essential transaction statuses safely.

---

##  Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Backend:** PHP (Procedural/Modular Server Logic)
* **Database:** MySQL (Relational Schema Design & Real-Time Querying)
